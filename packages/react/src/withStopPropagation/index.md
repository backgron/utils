# withStopPropagation

阻止组件事件冒泡

## API

```ts
withStopPropagation(<div>xxx</div>, ['click']);
```

### Params

| 参数    | 说明 | 类型         | 默认值 |
| ------- | ---- | ------------ | ------ |
| element | 元素 | ReactElement | -      |
| events  | 事件 | string[]     | -      |
