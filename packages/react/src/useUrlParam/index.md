# useUrlParam

获取 url 参数 hook

#### 例子

```ts
const id = useUrlParam('id');
```

#### 参数

| 参数 |              说明               |  类型  | 默认值 |
| :--: | :-----------------------------: | :----: | :----: |
| key  | 需要从 url params 上 获取的字段 | string |   -    |
