# attachPropertiesToComponent

把属性连接到 组件上

## API

```ts
attachPropertiesToComponent(Form, { Item, Group });
```

### Params

| 参数       | 说明       | 类型                | 默认值 |
| ---------- | ---------- | ------------------- | ------ |
| component  | 组件       | C                   | -      |
| properties | 连接的属性 | Record<string, any> | -      |
