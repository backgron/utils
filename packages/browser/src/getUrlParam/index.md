# getUrlParam

获取 url 参数

## API

```ts
getUrlParam('id'); //=> id
getUrlParam('name'); //=> null
```

### Params

| 参数 | 说明                        | 类型   | 默认值 |
| ---- | --------------------------- | ------ | ------ |
| key  | 需要从 url 上获取的参数字段 | string | -      |
