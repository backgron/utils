# getUrlParam

获取 url 参数

#### 例子

```ts
getUrlParam('id'); //=> id
getUrlParam('name'); //=> null
```

#### 参数

| 参数 |            说明             |  类型  | 默认值 |
| :--: | :-------------------------: | :----: | :----: |
| key  | 需要从 url 上获取的参数字段 | string |   -    |
