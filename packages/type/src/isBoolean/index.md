# isBoolean

检查输入值是否为 布尔值

## API

```ts
isBoolean(true); //=>true
isBoolean(false); //=>true
isBoolean(Object(true)); //=>true
isBoolean(Object(false)); //=>true
```

### Params

| 参数  | 说明   | 类型 | 默认值 |
| ----- | ------ | ---- | ------ |
| value | 输入值 | any  | -      |
